### Lancez l’application sur l’émulateur (API 26+ recommandé).
### Cliquez DÉMARRER SERVICE → notification apparaît + chrono tourne.
### Quittez l’application complètement → le service continue (testez en ouvrant le tiroir de notifications).
### Cliquez ARRÊTER SERVICE → tout s’arrête proprement.
